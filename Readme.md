# U.S. Championship V'ball

This is the emulation of V'Ball from Technos Japan ported to MiSTerFPGA by Pierre Cornier (Pierco).


## Games provided

- U.S. Championship V'ball (US)
- U.S. Championship V'ball (Bootleg of US set)
